Designed and developed a structured tribute website dedicated to Dr. A.P.J. Abdul Kalam. The project focuses on advanced CSS Layout techniques to create a professional, magazine-style interface without relying on external frameworks like Bootstrap.

Key Features:

Advanced Layouts: Implemented a CSS Grid system (grid-template-columns) to manage a responsive 2-column layout with a sidebar and main content area.

Flexbox Alignment: Utilized Flexbox for the header and navigation bar to ensure perfect vertical and horizontal alignment of logos and links.

Semantic Structure: Used proper HTML5 semantic tags to organize biographical sections, bibliographies, and image galleries.

Styling: Applied custom CSS for box-model spacing, typography, and visual hierarchy.

Tech Stack:

HTML5

CSS3 (Flexbox and Grid)
